Setup application
git clone https://github.com/azharjustin/Task-manager-Repo.git

Frontend
cd task-manager-app
npm install
npm run dev

Backend
cd backend
npm install
npm run dev

env file
uri = use your mongodb uri
port = your port
JWT_SECRET = your secret key
GOOGLE_CLIENT_ID = your google client id (google cloud console)
