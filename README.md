Setup application
git clone https://github.com/azharjustin/Task-manager-Repo.git

Frontend
1. cd task-manager-app
2. npm install
3. npm run dev

Backend
1. cd backend
2. npm install
3. npm run dev

env file
uri = use your mongodb uri
port = your port
JWT_SECRET = your secret key
GOOGLE_CLIENT_ID = your google client id (google cloud console)
